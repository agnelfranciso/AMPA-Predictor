How to Run the Engine

The AMPI is powered by the Agnel Match Prediction Engine (AMPE). You must run the engine to generate the data that powers this website.

Prerequisites

Before you begin, ensure you have Python 3.10 or higher installed on your system. You can verify this by opening a terminal and typing python --version.

Step-by-Step Setup

  • 1

    Locate ampe.bat

    Open the fifa-predictor folder in your file explorer.

  • 2

    Run the Engine Manager

    Double-click the ampe.bat file (or run it from your terminal). This powerful script will automatically:

    • Create a virtual environment
    • Install all required dependencies quietly
    • Launch a beautiful command-line menu
  • 3

    Generate Predictions

    From the AMPE menu, press 1 and hit Enter to Run Prediction Engine.

    Note: This process can take anywhere from 10 to 40 seconds depending on your machine and internet connection. Do not close the window until it finishes.
  • 4

    Success!

    Once the engine finishes, it will save the snapshot and return to the menu. You can then press 2 to launch the AMPI UI!
